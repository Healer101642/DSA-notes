# Arrays

## Characteristics
- Definition of arrays
- Basic properties (size, indexing, etc.)

## Time Complexity
- Access: O(1)
- Search: O(n)
- Insertion: O(n)
- Deletion: O(n)

## Operations
### Access
- Index-based access
### Search
- Linear search
- Binary search (if sorted)
### Insertion
- At end
- At specific index
### Deletion
- From end
- From specific index

## Real-world Applications
- Storing data in a collection
- Implementing other data structures (like heaps)

## Practice Problems
- Find maximum element in an array
- Rotate array by k positions
